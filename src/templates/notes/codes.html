{% trans %} Euro banknotes have two codes on them. One code is the printer code, it tells you about where the note was printed. The other code is the serial number. In the 2002 series of banknotes the serial number tells you which country issued the banknote, but in the Europa series it tells you where the note was printed, just like the printer code. {% endtrans %}

{{ _("Printer Code") }}

{% trans %} The printer code (not to be confused with the serial number) is a small code printed on banknotes with information about where the banknote was printed, along with other information. All printer codes take the form of ‘X000X0’, in other words a letter followed by 3 numbers, a letter, and a final number. {% endtrans %}

{% trans %} The printer code can be a bit tricky to find. The following dropdowns will show you where to find the printer code on each note. {% endtrans %}

{% trans %} The first letter in the printer code identifies the specific printer at which the banknote was printed. The tables below will tell you which letters correspond to which printers. The final letter and number form a pair (such as ‘A2’ or ‘D6’). This pair acts as a set of coordinates telling you where on the sheet of paper the banknote was located. During printing, banknotes will be printed multiple times on a large sheet of paper which is later cut into smaller individual banknotes. A note with the pair ‘A1’ will have been at the upper-left most corner of the printing sheet, with ‘A2’ to it’s right and ‘B1’ below it. {% endtrans %}
