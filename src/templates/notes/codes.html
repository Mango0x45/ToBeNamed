{% trans %} Euro banknotes have two codes on them. One code is the printer code, it tells you about where the note was printed. The other code is the serial number. In the 2002 series of banknotes the serial number tells you which country issued the banknote, but in the Europa series it tells you where the note was printed, just like the printer code. {% endtrans %}
